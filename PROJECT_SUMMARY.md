## üéØ WHAT IS QRQUEST?

**QRQuest** is a digital platform that transforms tourist attractions into discoverable, interactive destinations. Each tourist spot gets:

1. **Digital Profile** - Name, description, history, contact
2. **Multimedia Content** - Images and videos
3. **QR Code** - Scannable link to spot details
4. **Online Presence** - Accessible from web and via QR scan

### Real Example
Siddheshwar Temple in Solapur:
```
Admin Creates Spot
    ‚Üì
Upload image + description + video
    ‚Üì
System generates QR code
    ‚Üì
Visitor scans QR ‚Üí Sees temple details on phone
OR
Visitor visits spot.php?id=xyz ‚Üí Full information displayed
```

## üöÄ HOW IT WORKS

### Admin Workflow
```
1. Admin visits adminlogin.php
2. Enters email and password
3. System verifies bcrypt hash
4. Creates session if valid
5. Redirects to adminhome.php
6. Admin navigates to addspot.php
7. Fills spot form
8. Uploads image and video
9. System:
   - Generates unique ID (hex)
   - Creates QR code
   - Stores in database
   - Saves files
10. Spot is live!
```

### Visitor Workflow (Online)
```
1. Visitor visits website
2. Clicks on spot name
3. Gets spot.php?id=xyz
4. Sees:
   - Image
   - Description
   - Video
   - QR code
   - Contact info
```

### Visitor Workflow (QR Code)
```
1. Visitor at physical location
2. Scans QR code on sign
3. QR links to: spot.php?id=xyz
4. Phone opens link
5. Sees full spot details
6. Can access external links
7. Can call or email
```
